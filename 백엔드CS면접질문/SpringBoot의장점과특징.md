## Spring boot 의 장점과 특징에 대해 설명해주세요
Spring boot 의 장점으로는 Spring 프레임워크에서는 개발자가 직점 설정파일을 작성하고, 필요한 객체를 빈에등록하고 빈에대하 의존성 주입을 직접 시켜주었습니다.
하지만 Spring boot는 개발자가 설정파일을 직접 작성하지않고 프로젝트 설정,빈 객체 관리,  라이브러리 의존성 등을 자동으로 처리해주는 기능을 가지고 있습니다. 
spring boot의 특징으로는 톰캣과 같은 내장 서버가 있어 별도의 웹서버 설정이 없이 바로 실행이 가능합니다. 또, 스타터 의존성 패키지가 있어 라이브러리의 의존성을 자동으로 설정해줍니다. 마지막으로 설정 자동 기능을 통해 개발자가 일일히 설정을 하지 않아도 되는 특징이있습니다.

## Jpa랑 Mybatis차이가 뭔가요?
먼저 Mybais는 SQL을 자바 객체로 매핑해주는 프레임워크입니다 Mybatis의 특징으로는 개발자가 SQL문을 직접 작성하여 섬세한 테이블 데이터 관리를 할 수있습니다. JPA는 java의 ORM으로써 객체를 중심으로 SQL을 자동으로 만들어주고 객체와 매핑도 시켜주는 프레임워크입니다. JPA의 특징으로는 객체를 중심으로 작동하다보니 개발자가 비즈니스 로직에 좀 더 집중 할 수 있게 됩니다.

## JPA는 기본적으로 SQL을 자동으로 생성해 주는데, 만약 JPA를 기본 프레임워크로 사용하는 프로젝트에서 이러한 복잡한 동적 쿼리를 처리해야 한다면 지원자님은 어떤 기술이나 방식을 도입하여 이 문제를 해결하시겠습니까?

만약 JPA로 복잡한 동적 쿼리를 처리해야한다면 자바 객체와 메서드로 쿼리를 작성하는 QueryDSL로 동적쿼리를 처리 할것입니다.
