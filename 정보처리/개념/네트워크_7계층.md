# OSI 7계층

## 네트워크 계층(3계층)
IP 

- 데이터그램의 주소 지정 및 경로 설정

- 비연결성, 비신뢰성 

ARP

- IP주소를 MAC주소로 변환

- 동적, 브로드캐스트 기반 요청

RARP

- MAC 주소를 IP주소로 변환 

- ARP 역기능 부팅시 IP주소 할당시 사용 

ICMP

- 오류보고 및 네트워크 상태진단

- IP에 의존하여 동작, 제어 메시지 전송 

IGMP

- 멀티캐스트 그룹 관리 

- 특정 그룹에만 데이터 전송, 대역폭 효율화

라우팅 프로토콜 

- RIP, OSPF, BGP

NAT 

- 사설 IP 주소를 공인 IP주소로 변환 

### 데이터 링크 계층( 2계층)

HDLC 

비트 위주 프로토콜, 프레임 구조, 비트스터핑 

- 정보프레임(0부터 시작, 데이터전달)
- 감독프레임( 10 ~ 시작, 오류/흐름 제어)
- 비번호 프레임 (11~시작, 링크동작 모드 설정)

전송모드 

- 정규 응답 모드 :NRM
    - 종국에서 데이터 전송시 주국에서 허락 필요
- 비동기 균형 모드 : ABM
    - 양쪽에서 명령과 응답 모두 전송가능
- 비동기 응답 모드 : ARM
    - 불균형 모드/ 종국이 주국 허락없이 데이터 전송 및 송신 가능, 제어기능은 주국만 허용
